# savings-goal-calculator
Excel-based savings goal calculator dashboard for estimating periodic savings required to achieve a future financial target.
## Features

- Calculates required periodic savings

- Supports different contribution frequencies

- Displays total contributions and interest earned

- Interactive Excel dashboard layout

- Protected input and formula cells

## Skills Demonstrated

- Microsoft Excel

- Financial modelling

- Dashboard design

- Financial calculations

- Spreadsheet protection and user input controls
